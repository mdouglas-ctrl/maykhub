# Hub — Maykon Douglas (Mayk Lee)

Vitrine de produtos digitais. Site estático, sem build.

## Como editar
Todo o conteúdo está no topo de `index.html`, nos blocos `PERFIL` e `PROJETOS`.
- **PERFIL**: nome, frase de posicionamento (`tagline`), WhatsApp, Instagram, LinkedIn.
- **PROJETOS**: cada card é um bloco `{ }`. Copie um, cole abaixo e edite para adicionar.

Arquivos de projeto ficam em `/projetos`. Bibliotecas em `/vendor` (não mexer).

## Deploy
Conectado à Vercel. Cada commit nesta branch republica o site automaticamente.
